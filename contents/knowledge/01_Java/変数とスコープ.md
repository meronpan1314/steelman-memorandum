## 1. ローカル変数
- メソッド内で宣言
- 初期化必須


## 2. インスタンス変数
- クラスに属する
- デフォルト値あり


## 3. static変数
- クラスに1つ
- クラスロード時生成


## 4. スコープ

- ブロックスコープ
- for文内変数
- ラムダでの実質final


## 5. シャドーイング

```java
int x = 10;
{
    int x = 20; // コンパイルエラー
}
```

## 6. ライフサイクル

| 種類     | 生成     | 破棄     |
| ------ | ------ | ------ |
| ローカル   | メソッド呼出 | スタック解放 |
| インスタンス | new    | GC     |
| static | クラスロード | JVM終了  |
