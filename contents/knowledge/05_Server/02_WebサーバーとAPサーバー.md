## 役割分担

| 種別      | 主な役割             |
| ------- | ---------------- |
| Webサーバー | HTTP受付・静的コンテンツ配信 |
| APサーバー  | 動的処理・業務ロジック実行    |

---

## Webサーバー

### 主な役割

* HTTPリクエストの受付
* HTML / CSS / 画像の配信
* リバースプロキシ

### 代表例

* Nginx
* Apache HTTP Server

---

## APサーバー

### 主な役割

* 業務ロジック実行
* DBアクセス
* API提供

### 代表例

* Tomcat
* Spring Boot（組み込みAP）

---

## 3層構成

```
[ クライアント ]
      |
[ Webサーバー ]
      |
[ APサーバー ]
      |
[ DBサーバー ]
```

---

## まとめ

* Web：入口・振り分け
* AP：処理の中核
* 分離すると保守性・スケーラビリティが向上
